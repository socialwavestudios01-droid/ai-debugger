# AI Debugger

AI-powered debugging assistant for VS Code.

## Features

- 🤖 **Hello World** - Test command
- 📊 **AI Analyze Code** - Analyze selected code
- 🚀 **Status Bar Integration** - Quick access from bottom bar

## Usage

1. Select some code in your editor
2. Press `Ctrl+Shift+P`
3. Type "AI Analyze Code"
4. See the analysis result

## Requirements

- VS Code 1.85.0 or higher

## Extension Settings

This extension contributes the following commands:

* `ai-debugger.helloWorld` - Show hello message
* `ai-debugger.analyze` - Analyze selected code
