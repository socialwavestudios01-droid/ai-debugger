🔧 Quick Install

Method 1: VS Code Marketplace

Press Ctrl+Shift+X → Search "AI Debugger Pro" → Install

Method 2: Command Line

code --install-extension ai-debugger-0.0.1.vsix

Method 3: VSIX File

Download .vsix file → Extensions (Ctrl+Shift+X) → Install from VSIX
